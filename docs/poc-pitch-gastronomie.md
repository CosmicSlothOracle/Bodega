# Bloom OS — Digitales Betriebssystem für dein Restaurant

## Das Problem: Zu viele Tools, kein Überblick

Ein modernes Restaurant jongliert heute mit 5–8 verschiedenen Werkzeugen gleichzeitig:
WhatsApp-Gruppen für den Schichtplan, ein Excel für die Reservierungsliste, ein
externes Buchungssystem, ein POS-System, dazu Anrufe, Zettel, Missverständnisse.
Kein dieser Tools redet mit dem anderen. Fehler kosten Gäste — und Geld.

---

## Was Bloom OS ist

**Ein custom built zentrales Dashboard für Restaurantbetreiber und ihr Team** — läuft im Browser,
braucht keine Installation, funktioniert auf jedem Gerät.

Jeder Bereich des Tagesbetriebs ist dort abgebildet:

### Reservierungen
Alle Tischreservierungen an einem Ort. Online-Buchungen kommen automatisch rein.
Telefonische Reservierungen trägt der Mitarbeiter in Sekunden ein.
Kein Zettel, keine Excel-Tabelle, keine Doppelbelegungen.

### Schichtplanung
Der Owner oder Manager erstellt den Schichtplan direkt im Dashboard.
Mitarbeiter sehen ihren Plan sofort auf dem eigenen Gerät.
Tauschgesuche werden über das System gestellt, nicht per WhatsApp —
Owner bestätigt per Klick, alle Beteiligten werden automatisch informiert.

### Mitarbeiterkommunikation via Telegram
Kein privates WhatsApp des Chefs mehr. Benachrichtigungen über neue Schichten,
Tauschanfragen und Bestätigungen gehen automatisch als Telegram-Nachricht raus.
Wer kein Telegram will, bekommt eine E-Mail. Später noch whatsapp Schicht reminder 30 min vor Shcicht -> Mehrkosten nur als b2b solution verfügbar

### Analytics
Wie viele Gäste kamen wann? Welche Wochentage laufen am besten?
Woher kommen die Besucher der Website? Das Dashboard zeigt es automatisch —
keine manuellen Auswertungen, keine teuren Agenturen.

### Website inklusive
Die öffentliche Gastronomieseite (Speisekarte, Reservierung, Über uns, Galerie)
ist direkt integriert. Änderungen an Inhalten erscheinen sofort live.

| Schichtplanung | Im System, mit Benachrichtigung       | Nicht enthalten                     | WhatsApp-Chaos |

| Analytics | Automatisch, live                              | Teures Add-on                       | Manuell |

## Wie es für dein Team aussieht

**Owner/Chef (du):**
Öffnest das Dashboard — siehst sofort: Wer arbeitet heute, wie viele Reservierungen
stehen an, gibt es offene Tauschanfragen. Sonderwünsche von Gästen + Anmerkungen vom Owner zb. Augustiner ist leer -> Teganseer als Ersatz in Kühlschrank xy
 Alles auf einem Blick, von überall.

**Mitarbeiter:**
Bekommen einen Zugangslink per E-Mail. Öffnen das Dashboard im Browser.
Sehen ihren Schichtplan, können Tauschangebote stellen.
Keine App-Installation, kein Passwort das man vergisst.

**Gäste:**
Buchen wie gehabt online oder rufen an — erleben aber keine Änderung.
